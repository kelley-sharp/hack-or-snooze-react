# Hack or Snooze

Hack-or-Snooze is a clone of the Y-Combinator Hacker News site. 
You can sign up to create an account, post stories, and favorite stories. 
My first project exclusively using React hooks!

See the deployed version [here](https://ks-hack-or-snooze-react.netlify.app/).

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
